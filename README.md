🌐 Dhanu Portfolio — Personal Portfolio Website

A clean, responsive, and modern personal portfolio website built from scratch using HTML, CSS, and JavaScript. Designed to showcase my skills, projects, and professional profile as a Web Developer.

🔗 Live Site: Click here to view


📌 Sections


Home — Introduction with name, role, and quick links
Skills — Tech stack displayed as interactive pill tags
Projects — Cards showcasing my built projects
About — Brief bio and CV download
Contact — Social media and email links
